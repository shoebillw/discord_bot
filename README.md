## 機器人簡介

利用朋友的形象建立的discord機器人，可以祝人生日快樂，會直接在別人的留言區打麻將


## 指令簡介

>**文字類型指令:**

* prefix : '!'
* 指令 : 嗨/早安/晚安/冠廷/好油喔/冠廷幫我買遊戲/推薦的漫畫/拜託別煩我/天黑請閉眼/謝謝/打麻將/貓貓/柴柴/かわ余/冠廷生日快樂
* 指令用法 : !'指令'

>**音樂類型指令:**
* prefix : '@'
* 指令 : play/disconnect
* play指令用法 : @play 'youtube網址'
